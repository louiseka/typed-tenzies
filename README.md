# Typed Tenzies
A fun and interactive dice game built with React and TypeScript. Roll the dice until all of them show the same value. Click on individual dice to "hold" their values between rolls.

## Features 
- Built with TypeScript for type safety.
- Interactive dice that can be "held" between rolls.
- Detects when the player has won (all dice have the same value and are held).
- Fun Confetti animation when you win using react-confetti.
- Reset the game easily with a single button click.

## How to Play
- Click Roll to roll all dice.
- Click individual dice to hold their value.
- Continue rolling until all dice show the same number.
- Once all dice are held and show the same value, you win and confetti appears!
- Click New Game to restart.
